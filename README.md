# 🤖 AI Voice Calling Assistant

An AI-powered voice calling system that can talk to users over phone calls like a real human.  
It listens to user speech, understands it using AI, and responds with natural human-like voice.

---

## 🚀 Features

- 🎤 Real-time voice interaction  
- 🧠 AI-generated intelligent responses  
- 🔊 Human-like voice using Murf AI  
- 📞 Automated calling using Twilio  
- 🔁 Continuous conversation loop  
- ⚡ Fast and scalable  

---

## 🧠 How It Works

1. User receives or initiates a call  
2. System listens to user speech  
3. AI (Groq) processes the input  
4. Generates a smart response  
5. Murf AI converts text to speech  
6. Audio is played back to the user  
7. Conversation continues  

---

## 🛠️ Tech Stack

- **Groq API** – AI responses  
- **Murf AI** – Text-to-Speech  
- **Twilio** – Voice calling  
- **Node.js + Express** – Backend  
- **Ngrok** – Public URL

## ⚙️ Setup Instructions

### 1️⃣ Clone the repository

```bash
git clone https://github.com/your-username/ai-voice-calling-assistant.git
cd ai-voice-calling-assistant

2️⃣ Install dependencies
npm install

3️⃣ Create .env file

Create a file named .env in the root folder and add:

TWILIO_SID=your_twilio_sid
TWILIO_AUTH=your_twilio_auth
MURF_API_KEY=your_murf_api_key
GROQ_API_KEY=your_groq_api_key

4️⃣ Run the server
node index.js
